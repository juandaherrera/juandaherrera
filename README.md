## What I'm working on

At Snoonu, I'm the technical owner of several core platform services, including the ETA calculation service,
a personalized communications system built on AWS Personalize, and a centralized feature store for platform users.
My focus is on architecture, performance, production stability, and observability,
while ensuring seamless ML model integration across a scalable microservices stack.
